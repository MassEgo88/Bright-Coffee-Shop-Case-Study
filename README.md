# Bright-Coffee-Shop-Case-Study
This is where i as a Junior Data Analyst will be looking at the insights and analysing the sales of the business.

Project Overview

This project presents a data analysis of Bright Coffee Shop’s sales performance. The goal of this analysis is to uncover key business insights that support decision-making to:
•	Increase revenue 
•	Improve product performance 
•	Optimize daily operations 
The project focuses on identifying sales trends, peak hours, product performance, and areas for improvement.
________________________________________

Objectives

•	Analyze revenue across different store locations 
•	Identify peak and off-peak sales periods 
•	Evaluate product performance and best-selling items 
•	Understand weekday vs weekend sales behavior 
•	Provide actionable business recommendations 
________________________________________

Data System & Tools Used

This project follows a simple data pipeline:
1.	Raw Data Source 
o	Excel dataset containing transactional sales data 
2.	Data Processing 
o	SQL queries executed in Databricks 
o	Data cleaning and transformation 
o	Calculations: 
	Total sales 
	Time-based grouping (morning, afternoon, evening) 
3.	Data Storage 
o	Processed data stored in Databricks tables 
4.	Data Visualization 
o	Exported to Excel 
o	Charts and dashboards created for insights 
________________________________________

Key Insights

Revenue by Location
•	Revenue varies across store locations 
•	Some stores consistently outperform others 
Monthly Sales Trends
•	Sales increased steadily from January to June 
•	February showed lower performance compared to later months 
Time-Based Sales
•	Morning generates the highest sales (~50%) 
•	Afternoon contributes ~35% 
•	Evening contributes the least (~25%) 
Weekday vs Weekend
•	Weekdays generate more sales than weekends 
•	Weekends are underperforming and need improvement 
Product Performance
•	Coffee products are top-selling items 
•	Some products (e.g., packaged chocolate) underperform

Business Recommendations

1. Increase Sales During Off-Peak Hours
•	Introduce afternoon promotions 
•	Offer discounts and combo deals 
 2. Weekend Campaigns
•	Launch: 
o	Buy 1 Get 1 Free deals 
o	Group discounts 
o	Family offers 
3. Upselling Strategies
•	Create combo deals (e.g., coffee + pastry) 
•	Increase average order value 
4. Loyalty Programs
•	Implement loyalty cards or points system 
•	Encourage repeat customers 
5. Social Media Marketing
•	Promote: 
o	Happy hour deals 
o	Flash discounts 
•	Target younger audiences at low cost
